Goal of this exemple

RBF regression is a simple method to fit a function to a set of data. The goal of this exemple is to show how to use the RBF regression in the rbfregression package. the python package rbf-regression is available on gitlab.

To show how to use, we will use a synthetique data set. The data set is generated from a function with some noise. The goal is to fit a RBG regression model based on the data set.

The synthetique data to fit

The data aer generated from a "simple" function with some noise.

The function is:

$$ f(x) = \exp\left(-\left(\frac{x-50}{20}\right)^2\right)/2 + \frac{1}{150} \cdot x $$

The noise is a normal distribution with a standard deviation of $\sigma^2$ is added to the function.

The impementation of the function is:

In[9]:
import numpy as np
def function(x, s2 =.1):
    y = np.exp(-((x-50)/20.)**2)/2 + 1/150.*x
    y += s2 * np.random.randn(len(x))
    return y

plot the data

Two set of points are created :

  • the training set with noise
  • the test set without noise that will be used to evaluate the ground function

RBF kernel definition

the equation of the radial basis function kernel is given by: $$ k(x_i, x_j) = \exp\left(-\frac{d^2}{2 r^2}\right) $$ where $r$ is the kernel bandwidth, and $d$ is the euclidian distance between the two points $x_i$ and $x_j$.

In[15]:
def Gauss_kernel(d, radius):
    radius *= 0.707
    result = np.exp(-(d **2 ) / (2.0 * radius**2)) 
    return result

The RBF kernel can be plotted for different values of $r$:

Fit the RBF regression

Once the RBF kernel is defined, the RBF regression can be fitted to the data set.

A set of $N$ key points is defined. This key point can be seen as control points. It means that the RBF regression model will be constructed using $N$ RBF kernels centered on the $N$ key points.

In[30]:
# Key points
X_k = np.linspace(10, 141, 8) 
X_k = np.atleast_2d(X_k).T
In[32]:
# import of the rbfregression module
import rbfregression.core as rbfreg
# Create the RegularizedRBF instance
RBF = rbfreg.RegularizedRBF(kernel=Gauss_kernel)

# Set the control points (key points)
RBF.set_key_points(X_k)

# Compute the fit
RBF.fit(X_l, Y_l, regularization=0.0, radius = 25)

# Eval the model at points
Y_s = RBF.eval(X_l)
Y_k = RBF.eval(X_k)

# Plot them all
